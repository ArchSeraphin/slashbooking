Prenez des rendez-vous sur votre WordPress

SlashBooking est un plugin WordPress open source qui ajoute un calendrier de réservation à votre site, synchronisé en temps réel avec votre Google Calendar. Vos clients réservent, vous validez, et tout reste dans votre infrastructure.

Fonctionnalités principales

Tout ce qu'il faut pour gérer vos rendez-vous, rien de superflu.

📅

Calendrier visuel

Un calendrier mensuel avec créneaux disponibles, partiels, complets ou fermés affichés en un coup d'œil pour vos visiteurs.

🔄

Sync Google Calendar bidirectionnelle

Les RDV créés sur le formulaire apparaissent dans votre Google Calendar. Les événements que vous créez dans Google bloquent automatiquement les créneaux.

✉️

E-mails transactionnels personnalisables

6 templates HTML modifiables depuis le panneau admin (réception, confirmation, refus, annulation, rappel J-1) avec aperçu live.

👍

Validation 1-clic

Validez ou refusez les demandes en cliquant directement dans l'e-mail de notification. Liens signés HMAC, expirent en 72h.

⚙️

Services configurables

Durée, jours et plages horaires d'ouverture, délai minimum de prise, horizon de réservation — tout se règle par service depuis l'admin.

🛡️

Anti-spam Cloudflare Turnstile

Protection optionnelle du formulaire public contre les bots, sans capture de données personnelles ni casse-tête CAPTCHA.

🇪🇺

RGPD-compliant

Données auto-hébergées sur votre WordPress. Connecteurs d'export et d'effacement WP intégrés. Rétention configurable.

🔐

Tokens chiffrés

Les refresh tokens Google sont stockés chiffrés (sodium) avec une clé hors base de données. Aucune donnée Google n'est exfiltrée vers un service tiers.

Installation

  1. Téléchargez le fichier .zip ci-dessus.
  2. WordPress Admin → Extensions → Ajouter → Téléverser.
  3. Activez le plugin. Six tables wp_sb_* sont créées automatiquement.
  4. Configurez votre Google Calendar dans SlashBooking → Google.
  5. Insérez le shortcode [slashbooking] dans n'importe quelle page.

Comment ça marche

Un parcours utilisateur sobre, conçu pour la conversion.

1

Le visiteur choisit un créneau

Sur votre site WordPress, le visiteur ouvre le calendrier visuel, sélectionne un jour disponible, puis un créneau.

2

Il remplit le formulaire

Nom, e-mail, téléphone, adresse, notes éventuelles. Consentement RGPD requis. Anti-bot Turnstile si activé.

3

Vous recevez un e-mail

Un e-mail vous notifie immédiatement. Vous validez ou refusez d'un clic, ou utilisez le tableau de bord admin.

4

Sync automatique

À la validation, l'événement est créé dans votre Google Calendar. Le client reçoit la confirmation avec un fichier .ics joint.
